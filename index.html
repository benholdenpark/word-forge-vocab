💡 모바일 사용 TIP

매번 주소를 입력하거나 링크를 찾는 대신, 휴대폰 바탕화면에 아이콘을 만들어두면 앱처럼 한 번 터치로 바로 실행할 수 있어서 훨씬 편해요.

🤖 안드로이드 (크롬 브라우저)
  1. 크롬 브라우저로 이 페이지를 엽니다
  2. 오른쪽 위 점 세 개(⋮) 메뉴를 누릅니다
  3. "홈 화면에 추가"를 선택합니다
  4. 이름을 확인하고 "추가"를 누릅니다
🍎 아이폰 (사파리 또는 크롬)
  1. 사파리(Safari) 또는 크롬(Chrome) 중 쓰시는 브라우저로 이 페이지를 엽니다
  2. 화면 아래(또는 위쪽) 공유 버튼(네모 안에 위쪽 화살표 ⬆️)을 누릅니다
  3. 아래로 스크롤해서 "홈 화면에 추가"를 찾아 누릅니다
  4. 이름을 확인하고 오른쪽 위 "추가"를 누릅니다

(참고: 크롬은 비교적 최신 버전에서만 이 기능이 지원돼요. 혹시 "홈 화면에 추가" 메뉴가 안 보이면 사파리로 시도해보세요.)
